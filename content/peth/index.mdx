# Polaris pETH and Bonding Curve

pETH is ETH deposited into the Polaris bonding curve. It is not a CDP-minted pAsset; it is the collateral token that backs pAssets.

<Callout variant="info" title="Start here">
If you are trying to get pETH in the app, start with [Swap](/using-app/swap). If you are trying to understand fpETH, vpETH, or reserve loans, start with [pETH Split](/peth/split).
</Callout>
