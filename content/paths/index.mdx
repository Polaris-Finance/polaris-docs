# Polaris User Paths

Start with the path that matches the action you want to take, but check [Launch Status](/launch-status) first. Polaris is currently public Public Testnet 1 on Sepolia; production mainnet actions are not yet live.

## Start here if

| If you want to... | Read |
|---|---|
| Verify the official app, network, addresses, and audit status | [Polaris Launch Status](/launch-status), then [Safety and Verification](/paths/safety-verification) |
| Mint pUSD, pGOLD, or another pAsset | [Minting & Troves](/minting) |
| Earn pAsset or pETH-denominated yield | [Earning Yield](/yield) |
| Hold pETH or understand the bonding curve first | [pETH & Bonding Curve](/peth) |
| Convert pETH, stake POLAR, or study future vePOLAR locking | [POLAR & Staking](/polar) |
| Build an app, indexer, vault, or risk system | [Develop on Polaris](/paths/develop) |

## Actionability

These paths are decision guides, not production launch instructions until [Launch Status](/launch-status) lists the official app, production contracts, final parameters, and audit status. For now, builders can plan integrations and test on Sepolia with testnet manifests.
