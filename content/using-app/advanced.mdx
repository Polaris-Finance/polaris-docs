# Advanced

<PageStatusBanner lastUpdated="June 2026" />

Use **Advanced** for tools that expose protocol mechanics more directly than the main task tabs. These surfaces are useful for power users and integrators, but they can bypass the guardrails of a guided flow.

<Callout variant="warning" title="Where this appears in the app">
App anchor: **Advanced** tab. Use this surface only when you know the exact contract, function, token, amount, and effect you intend to sign.
</Callout>

## Common Advanced tools

| Tool | What it is | Read first |
|---|---|---|
| **Approvals** | Token allowances for app contracts | [Safety and Verification](/resources/safety-verification) |
| **PSM** | Peg Stability Module actions for a pAsset instance | [Redemptions](/redemptions-liquidations) |
| **Reserve Loans** | WETH loans against fpETH floor-value collateral | [pETH Split](/peth/split) |
| **POLAR Staking** | Stake, unstake, and claim routed rewards where exposed | [POLAR Tokenomics](/polar/tokenomics) |
| **Conversion** | Burn pETH to mint POLAR through the auction | [Conversion Auctions](/polar/conversion-auctions#how-to-convert) |
