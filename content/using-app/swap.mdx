# Swap

<PageStatusBanner lastUpdated="June 2026" />

## What it does

Swap exchanges app-supported assets through the route quoted by Polaris. Routes may include WETH, pETH, fpETH, vpETH, pAssets, or protocol modules exposed by the current Public Testnet 1 deployment.

![Polaris Swap tab showing input and output assets, quoted route, minimum received, and slippage controls](/screenshots/swap.jpg)

*Swap quotes should be refreshed and reviewed before approval or execution.*

## Before you start

- Use [app.testnet.polarisfinance.io](https://app.testnet.polarisfinance.io) on Sepolia, chain ID `11155111`.
- Hold the input token and enough Sepolia ETH for gas.
- Read the route, output, minimum received, price impact, and approval target before signing.

## Steps/checks

1. Choose the input and output assets.
2. Enter the amount.
3. Review the route and quote.
4. Check minimum received, price impact, fees, and slippage.
5. Approve the token only if the spender matches the app flow.
6. Sign the swap transaction.
7. Confirm the output balance appears in the wallet and Dashboard.

The swap worked when the transaction lands and the output token balance updates.

## Common failures

- Quote expired before the transaction landed.
- Slippage limit was too tight.
- Input balance or allowance was too low.
- Gas was too low or Sepolia was congested.
- Wallet is connected to the wrong account or network.
- The route changed after refresh.

## Risks/links

Swaps can lose value through slippage, price movement, stale quotes, gas, and execution ordering. pETH and pAsset mechanics are explained in [pETH](/peth), [pETH Floor Price](/peth/floor-price), and [pAsset Catalog](/minting/passet-catalog). Use [Risk Disclosure](/resources/risk-disclosure) for risk details.
