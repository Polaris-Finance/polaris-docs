# Earn

<PageStatusBanner lastUpdated="June 8, 2026" />

## What it does

Earn groups protocol-native yield actions exposed in the app. This includes Stability Pool deposits, withdrawals, claimable gains, and POLAR staking controls where available in the current deployment.

![Polaris Earn tab showing Stability Pool deposits, staking controls, claimable rewards, and APR context](/screenshots/earn.jpg)

*Earn groups deposit, withdrawal, claim, and staking actions in one place.*

## Before you start

- Use [app.testnet.polarisfinance.io](https://app.testnet.polarisfinance.io) on Sepolia, chain ID `11155111`.
- Hold the deposit token and Sepolia ETH for gas.
- Check whether you are depositing into a Stability Pool, staking POLAR, or claiming rewards.
- Read [Stability Pool](/yield) before depositing.

## Steps/checks

1. Select the Earn surface.
2. Check deposit balance, current deposit, APR estimate, and pending gains.
3. Choose deposit, withdraw, claim, or stake/unstake.
4. Review whether the action also claims pending rewards.
5. Approve the token if required.
6. Sign the transaction.
7. Confirm Dashboard balances, deposits, and claimable gains updated.

The action worked when the Earn card and wallet balances reflect the transaction.

## Common failures

- Deposit token balance is too low.
- Approval was signed but the action transaction was not.
- Pending rewards are dust-sized or not claimable yet.
- Withdrawal or claim is disabled by current app state.
- APR changed after refresh.
- Wallet is connected to the wrong account.

## Risks/links

Stability Pool deposits are first-loss capital during liquidations. APR estimates are not guaranteed returns. Use [Yield Sources](/yield/yield-sources), [POLAR Tokenomics](/polar/tokenomics), and [Risk Disclosure](/resources/risk-disclosure) for the underlying risk model.
