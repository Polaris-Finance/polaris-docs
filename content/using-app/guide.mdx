# Guide

Use **Guide** when you know what outcome you want but not which app tab or docs page to start with.

## Choose by goal

| Goal | App tab | Docs path |
|---|---|---|
| Get pETH | Swap | [Hold or Use pETH](/paths/hold-use-peth) |
| Mint pUSD or pGOLD | Borrow | [Borrow pAssets](/paths/borrow-passets) |
| Deposit into the Stability Pool | Earn or Zap | [Earn Yield](/paths/earn-yield) |
| Split pETH into fpETH and vpETH | Split | [pETH Split](/peth/split) |
| Convert pETH into POLAR | Advanced or conversion panel | [Convert / Stake POLAR](/paths/convert-lock-polar) |
| Verify before interacting | Dashboard, then any action tab | [Safety and Verification](/paths/safety-verification) |

## How to use it

- Start with the goal card that matches your intent.
- Read the linked docs before signing.
- Return to the app tab for the actual transaction quote and simulation.

The Guide can route you to a flow, but it cannot verify that a quote is fresh or that a contract address is official. Do those checks in the action tab and against [Launch Status](/launch-status).
