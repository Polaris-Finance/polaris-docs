# Polaris Glossary

Use this page when a term in the app, docs, or contracts needs a precise meaning. User-facing pages prefer the simpler public labels below.

Also in Reference: [Public Testnet 1](/resources/testnet) · [Audits & Security](/resources/audits-security) · [Risk Disclosure](/resources/risk-disclosure) · [FAQ](/resources/faq) · [Brand Assets](/resources/brand-assets)

## Public Terms

**Advanced**
The app area for expert operations: PSM issue/redeem, pETH-to-POLAR conversion, and the testnet faucet. See [Advanced](/using-app/advanced).

**Borrow**
The app tab where users issue pAssets against pETH or borrow ETH against fpETH. See [Borrow](/using-app/borrow).

**Earn Vault**
A pAsset deposit venue that earns from borrower interest and liquidation gains. Earn Vaults are first-loss backstops, so deposits can be consumed during liquidations. See [Earn Vaults](/yield).

**fpETH**
The floor-price leg created when pETH is split. It represents pETH floor exposure and can be used for ETH borrowing strategies. See [pETH Split](/peth/split).

**LTV**
Loan-to-value: debt value divided by collateral value. Lower LTV gives more room before liquidation. See [Manage a Position](/minting/managing-your-trove).

**pAsset**
A Polaris synthetic asset backed by pETH collateral, such as pUSD or pGOLD. See [pAsset Catalog](/minting/passet-catalog).

**pETH**
The ETH-backed reserve and collateral asset created through the Polaris bonding curve. See [pETH](/peth).

**pGOLD**
A pAsset designed to track gold exposure without issuer custody at the protocol layer. See [pAsset Catalog](/minting/passet-catalog).

**POLAR**
The stewardship token used for long-term protocol coordination and value capture. See [POLAR](/polar).

**Position**
An individual borrow position backed by pETH collateral. Users can add collateral, issue more, repay, withdraw collateral, or close it. See [Issuing pAssets](/minting).

**pUSD**
The flagship Polaris pAsset, designed to track the US dollar and backed by pETH collateral.

**Redemption**
Exchanging a below-peg pAsset for pETH collateral at protocol reference value minus a fee. See [Redemptions](/redemptions-liquidations).

**Split**
The operation that turns pETH into equal amounts of fpETH and vpETH, or merges equal fpETH and vpETH back into pETH. See [Split](/using-app/split).

**vpETH**
The volatility-premium leg created when pETH is split. It represents exposure to pETH value above the floor.

**Zap**
An app flow that bundles several actions into one transaction. A zap simplifies execution, but it does not change the underlying economics. See [Zap](/using-app/zap).

## Technical Aliases

These names may appear in contract labels, older materials, or internal implementation details. Public docs should prefer the terms above.

**CDP**
Legacy and contract-facing label for an individual borrow position. Public docs use **position**.

**CDB / branch**
Legacy and contract-facing label for a pAsset market. Public docs use **pAsset market** or **protocol market**.

**ICR / MCR**
Collateral-ratio terms used in some protocol references. They are the inverse of LTV: collateral value divided by debt value.

**Mint / issuing**
Contract-facing language for creating a token. Public docs use **issue**, **create**, or **receive** depending on the action.

**PSM**
Peg Stability Module. The direct mechanism for issuing or redeeming pAssets against pETH at the oracle price.

**SP / Earn Vault**
Internal name for the liquidation backstop behind an Earn Vault. Public docs use **Earn Vault**.

**Position**
Legacy label for an individual borrow position. Public docs use **position**.
