# Polaris Contracts & Addresses

> **Production note:** Canonical production addresses publish at deployment, not before. The contract *list* below reflects the intended production deployment shape; the **Address** column fills in at launch, each address linking to verified source on the official explorer. Current Sepolia addresses and parameters are testnet-only and live on [Public Testnet 1](/resources/testnet). Treat any production address not linked from these docs as unverified. See [Polaris Launch Status](/launch-status) for the current network state.

## How the system deploys

Polaris is a *factory* of pAsset instances. A single **shared core** — the bonding curve, POLAR, the Fee Router, and the oracles — deploys **once**. Then **one instance set** deploys **per pAsset** (pUSD, pGOLD, …): each gets its own CDP manager, Stability Pool, and internal Peg Stability Module, but all share the same pETH collateral, bonding curve, and POLAR.

So the address table has a **core section** plus **one block per pAsset**. The current docs focus on **pUSD** and **pGOLD**; production deployment status remains governed by [Launch Status](/launch-status), while current Sepolia artifacts are listed on [Public Testnet 1](/resources/testnet).

## Per-pAsset instance

One set deploys per pAsset. Key everything by `(pAsset, owner)` — a pUSD trove and a pGOLD trove are unrelated positions in different instances.

| Contract | Source (`src/`) | Address |
|---|---|---|
| **pAsset token** (e.g. pUSD, pGOLD) — the synthetic pegged ERC-20. | `StableToken.sol` | _Forthcoming_ |
| **CDP / Trove manager** — trove accounting, mint/redeem, liquidations, interest, Emergency Mode. | `CDPManager.sol` | _Forthcoming_ |
| Trove NFT — tokenizes each trove position. | `CDPNFT.sol` | _Forthcoming_ |
| [Stability Pool](/yield) — absorbs liquidations; the primary yield venue. | `StabilityPool.sol` | _Forthcoming_ |
| Collateral Surplus Pool — holds post-liquidation collateral surplus for borrowers to claim. | `CollSurplusPool.sol` | _Forthcoming_ |
| **Internal Peg Stability Module** — mint/redeem the pAsset against pETH; the peg and [redemption](/redemptions-liquidations) entry point. This is not a custodial or external-collateral PSM. | `PegStabilityModule.sol` | _Forthcoming_ |
| Interest controller — primary (autonomous) and secondary (utilization) interest rates. | `InterestRate/InterestController.sol` | _Forthcoming_ |
| Primary interest router — splits borrower interest across the Stability Pool, POLAR staking, and ecosystem. | `PrimaryInterestRouter.sol` | _Forthcoming_ |
| Price feed — the pAsset's reference price path. | `PUSDTestnetPriceFeed.sol` / `PGOLDTestnetPriceFeed.sol` | _Forthcoming_ |
| Oracle (medianiser) — median of multiple ETH/USD feeds (plus XAU/USD for pGOLD). | `Medianiser.sol` | _Forthcoming_ |

> **Read helpers** (`Helpers/MultiCDPGetter.sol`, `Helpers/MultiReserveLoanGetter.sol`) are view contracts for batched reads; integrations may use them but should treat direct contract reads as the source of truth.

> **A note on naming.** "pETH" and "the bonding curve" are the **same contract**. The "conversion auction" is **part of POLAR**, not a separate contract. The per-pAsset internal Peg Stability Module is the pETH mint/redeem path, not the custodial/external-collateral PSM model Polaris rejects. The "vePOLAR" lock layer is **not yet production-deployed**; production yield and staking details are pending. `pCHF`, `pBigMac`, and other catalog entries are illustrative; **pUSD** and **pGOLD** are the current Public Testnet 1 focus.

## Before you interact

1. Confirm the **network and chain ID** match the official deployment manifest.
2. Confirm every **address** matches [Public Testnet 1](/resources/testnet) for Sepolia or this page for production once populated.
3. Confirm the **ABI** matches that verified source.
4. Never paste an address from a screenshot, DM, or search result without that explorer check — address-spoofing is how these attacks succeed.

While production addresses are pending, see [Audits & Security](/resources/audits-security) for the security model, [Public Testnet 1](/resources/testnet) for Sepolia artifacts, or the [Glossary](/resources) for what each contract does.
