# Polaris Finance Documentation

<PageStatusBanner lastUpdated="June 8, 2026" />

Polaris produces **pETH** and uses it as base collateral for synthetic assets (**pAssets**) minted through onchain debt positions.

This documentation covers how the system works, how to use the app, and which risks to check before signing. For current network, contract, and actionability status, start with [Launch Status](/launch-status).

## The three engines

Polaris has three engines:

| Engine | What it is | Why it matters |
|---|---|---|
| **1.&nbsp;pETH** | ETH deposited into a two-way **bonding curve**, with a contract-defined floor under stated assumptions | A capital-efficient, yield-bearing collateral with a contract-defined exit path |
| **2.&nbsp;pAssets** | Synthetic pegged (USD, Gold, etc.) assets minted against collateral, via a Collateralized Debt Position (CDP) | Counterparty-free stables that scale with onchain activity |
| **3.&nbsp;POLAR** | The stewardship token, minted only by burning pETH in a **conversion auction** | Aligns long-term incentives and steers the protocol within hardcoded bounds |
